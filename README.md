MedRAG is a Multi-Modal Retrieval-Augmented Generation system developed to enhance medical information retrieval and reasoning. The platform extends beyond traditional text-centric RAG pipelines by incorporating heterogeneous healthcare data sources such as medical publications, radiological images, diagnostic reports, and anatomical diagrams. Through unified retrieval and generation mechanisms, the system delivers evidence-grounded responses that leverage both textual and visual medical knowledge, improving the quality and reliability of AI-assisted healthcare interactions.
